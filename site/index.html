Littoral

A TIDAL desktop player that exposes a local REST + WebSocket API — control playback and receive real‑time events from any application.

↓  Download for Windows View on GitHub

Requires a TIDAL HiFi or HiFi Plus subscription

Features

🎵

Full playback control

Play, pause, skip, seek, adjust volume and manage the queue — all accessible through a clean HTTP API.

⚡

Real‑time WebSocket events

Subscribe to now-playing, playback-state and position events. Build dashboards, stream overlays or automations on top.

🔎

Catalogue search

Search tracks, albums, artists and playlists via the API and enqueue results directly from any external app or script.

🖥️

Windows media integration

Track info and artwork appear on the lock screen, in the taskbar media widget and in Xbox Game Bar — no setup required.

🎮

Discord Rich Presence

Shows the current track in your Discord status as a "Listening to…" activity. Works out of the box.

📄

Interactive API docs

A Swagger UI is served locally at 127.0.0.1:7143/docs — explore and test every endpoint without leaving your browser.

REST API — quick look

http://127.0.0.1:7143
GET /now-playing Current track, state and position
POST /playback/toggle Play / pause
POST /playback/next Skip to next track
POST /playback/seek Seek to a position in seconds
POST /playback/volume Set volume (0–100)
GET /queue Upcoming tracks
POST /queue/enqueue Add a track to the queue
GET /catalogue/search Search TIDAL catalogue
GET /docs Interactive Swagger UI
